

#  Wonderland: Сеть Хоррор-Квестов

Проект представляет собой платформу для бронирования и управления хоррор-квестами. 
Разработан в рамках дипломной работы. 

Архитектура проекта — гибридная:
* **Backend:** Ruby on Rails (API, админ-панель, бизнес-логика, генерация расписания, финансы).
* **Frontend:** React + Vite + Tailwind CSS + Framer Motion (клиентская витрина).
* **База данных:** PostgreSQL.
* **Распознавание чеков/бланков (ИИ):** Tesseract OCR + MiniMagick.
