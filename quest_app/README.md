# Quest Management & Analytics System (QMAS)

Веб-ориентированная информационная система управления, автоматизации и аналитики для сети квест-румов. Проект разработан в рамках выпускной квалификационной работы по направлению «Прикладная математика и информатика».

Система оптимизирует внутренние бизнес-процессы (бронирование, расписание, учет выручки) и внедряет три интеллектуальных модуля на стыке интеллектуального анализа данных и математического моделирования.

## Ключевой функционал и математические модули

1. **Интеллектуальная рекомендательная система (Recommender Engine):**
   * Гибридный алгоритм, сочетающий контентную фильтрацию (Content-Based) на основе жанровых тегов и коллаборативную фильтрацию (Collaborative Filtering) по косинусному сходству оценок пользователей.
   * Эффективно решает проблему «холодного старта» для новых пользователей за счет динамического переключения режимов.

2. **Модуль OCR для распознавания бланков отчетности:**
   * Автоматическая обработка изображений чеков и отчетов с помощью библиотек `MiniMagick` и `RTesseract` (движок Tesseract OCR).
   * Предварительная цифровая обработка: линейное контрастирование, бинаризация и нормализация изображения для повышения точности распознавания.
   * Парсинг неструктурированного текста с использованием регулярных выражений (RegEx) для извлечения сумм, скидок и количества игроков.

3. **Динамический генератор дискретного расписания:**
   * Алгоритм автоматического формирования сетки сеансов с шагом в 90 минут.
   * Валидация пересечений, фильтрация прошедших слотов в реальном времени и автоматический расчет доступности комнат.

4. **Модуль геймификации персонала (Wonderpoints):**
   * Система мотивации актеров и операторов на основе KPI, штрафов и поощрений.
   * Динамическое распределение призового пула (Tier-1, Tier-2, Winner) по результатам игровых смен.

## Стек технологий

### Backend (API)
* **Фреймворк:** Ruby on Rails 8.1 (Режим API)
* **Язык:** Ruby 3.3+
* **База данных:** PostgreSQL 14
* **Обработка изображений / OCR:** MiniMagick, Tesseract OCR (RTesseract)

### Frontend (SPA)
* **Язык:** JavaScript (ES6+) / React 18
* **Сборщик:** Vite
* **Стилизация:** Tailwind CSS
* **Анимации:** Framer Motion
* **UI/UX Эстетика:** Кастомный темный интерактивный интерфейс в стиле Dark Gothic / Cyber Horror (шрифты *Nosifer*, глитч-эффекты, цветовые палитры *blood* и *void*).

### DevOps & Инфраструктура
* Контейнеризация приложения с помощью **Docker** и **Docker Compose**.
* Настроенный CI-пайплайн (**GitHub Actions**) для автоматической проверки безопасности кода (`Brakeman`, `bundler-audit`).

## Архитектура проекта

Проект реализован в виде архитектуры разделенного фронтенда и бэкенда (Decoupled Architecture):
* `/quest_app` — исходный код бэкенда на Ruby on Rails.
* `/frontend-quest` — исходный код клиентского приложения на React.
* `docker-compose.yml` — конфигурация оркестрации контейнеров.
